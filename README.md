# Features:
1. Dynamic reordering
2. Drag and drop (Dragula.js)
3. Animated confetti explosions 
4. Donut chart (Chart.js)
5. Countdown timer and alarm trigger
6. Dark and light mode toggle
7. Database: MongoDB
